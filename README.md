# apabhramsic_accent

pc_aclist.txtは『パウマチャリウ』の二句連詩行の句末に出現する全ての語について、脚韻が開始する箇所を示したファイルである。/後の母音から脚韻が開始することを表す。


accent_sorted.csvは各語についてアパブランシャ語の強勢を復元したデータである。各行は、番号、アパブランシャ語の語形、復元される強勢の位置、意味、脚韻開始一致率、内訳、全体、逸脱、ヴェーダ・サンスクリットのアクセントを表す。復元される強勢の位置は、最多の出現頻度の脚韻の開始位置をもとにしている。脚韻開始一致率とは、該当する脚韻の開始位置の出現頻度である。内訳は、最多の出現頻度の脚韻の開始位置の用例の数であり、全体は、その語の用例の総数である。逸脱は、最多の出現頻度の脚韻の開始位置よりも語末に近い位置から脚韻が開始する用例の数である。最多の出現頻度の脚韻の開始位置が複数ある場合は、逸脱のセルを空にした。ヴェーダ・サンスクリットのアクセントが記録されない語は、アクセントなしでサンスクリットの語形を記した。サンスクリットに対応する語が存在しない場合は、ヴェーダ・サンスクリットのアクセントのセルを空にした。

アパブランシャ語の各語は、名詞に関してはKumāra (1987)のエントリーに従った。代名詞は各屈折形をエントリーとした。動詞は現在語幹、希求法語幹、未来語幹、受動語幹、使役語幹、過去分詞、不定詞、絶対分詞、動形容詞を別のエントリーとした。


『パウマチャリウ』の電子テキストは、Eva de ClercqがBhayani (1953-1960) を底本に作成した電子テキストを用いた。以下のリンクから入手可能である。
https://gretil.sub.uni-goettingen.de/gretil/2_prakrt/spaucaru.htm

開音節における<ĕ>, <ŏ> は /i/ , /u/ の異音であるため、これらは<i>, <u> によって置換している。


参考文献
